## 설명
###  기본 동작
#### ESP8266을 사용하여 수집한 온/습도,미세먼지 정보를 LCD디스플레이와 웹사이트에 표시함
### 기능
#### 1. 수집 된 데이터를 DB에서 받아 Progress Bar에 실시간으로 표시
#### 2.  수집 된 데이터를 활용하여 미세먼지 단계(WHO 기준), 불쾌지수(기상청 공식 대입), 온도 단계(기상청 자료 기준) 를 표시
#### 3. 기기 온/오프라인 표시
